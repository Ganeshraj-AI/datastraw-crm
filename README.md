# Customer Support Ticket CRM

A full-stack Customer Support Ticket Management System built using FastAPI, SQLite, HTML, Tailwind CSS, and Jinja2.

## Features

- Create Support Tickets
- Search Tickets
- Filter Tickets by Status
- Update Ticket Status
- Add Notes/Comments
- Ticket Details View

## Tech Stack

Backend:
- FastAPI
- SQLAlchemy
- SQLite

Frontend:
- HTML
- Tailwind CSS
- Jinja2

Deployment:
- Railway

## Run Locally

1. Clone repo
2. Install requirements
3. Run:

uvicorn main:app --reload
